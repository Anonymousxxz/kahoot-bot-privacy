Kahoot Bot — Chrome Extension

Privacy Policy

Last updated: April 2025  ·  Version 1.0

Short version: This extension does not collect, store, transmit, or share any personal data. All processing happens locally in your browser. No analytics, no tracking, no servers owned by this extension.

What this extension does

Kahoot Bot is a browser extension that automatically detects quiz questions on kahoot.it and interacts with the page on your behalf using the Chrome DevTools Protocol (CDP). It uses an external AI API (Groq) solely to determine the correct answer to display.

Data we do NOT collect

  • No personal information (name, email, account credentials)
  • No browsing history outside of kahoot.it
  • No keystroke logging or clipboard data stored
  • No analytics, telemetry, or usage statistics
  • No cookies set by this extension

Data sent to third-party APIs

When a question is detected on kahoot.it, the extension sends the following to the Groq API (api.groq.com) to obtain an answer:

  • The text of the quiz question currently displayed on screen
  • The text of the available answer options
  • Optionally, the URL of an image embedded in the question

This data is used only to generate a response and is subject to Groq's own Privacy Policy. No personal identifiers are included in these requests.

Permissions used and why

Permission Reason
debugger Required to dispatch trusted mouse and keyboard events via Chrome DevTools Protocol so that clicks register as real user input on kahoot.it.
tabs Used to identify the active tab ID in order to attach the debugger to the correct tab.
activeTab Limits extension access to only the tab the user is currently viewing.
scripting Used to inject the content script into kahoot.it pages.
storage Reserved for storing user preferences locally in the browser. No data is sent externally.

Local storage

The extension may use chrome.storage.local to save configuration preferences (such as whether the bot is enabled). This data never leaves your device.

Children's privacy

This extension is not directed at children under the age of 13. We do not knowingly collect any information from children.

Changes to this policy

If this policy is updated, the "Last updated" date at the top of this page will reflect the change. Continued use of the extension after any changes constitutes acceptance of the updated policy.

Contact

If you have questions about this privacy policy, you can open an issue on the extension's repository or contact the developer directly through the Chrome Web Store listing.
